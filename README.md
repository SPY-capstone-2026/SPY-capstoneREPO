# SPY-capstoneREPO

<a id="team-17"></a>
## Team 17 SPY

| 항목 | 내용 |
|------|------|
| 프로젝트명 | AI를 활용한 유튜브 댓글 기반 인터넷 밈 트렌드 분석 플랫폼 |
| 서비스명(브랜드) | |
| 트랙 | 산학 |
| 팀명 | SPY |
| 팀구성 | 신희조, 윤수연, 박은수 |
| 팀지도교수 |  |
| 무엇을 만들고자 하는가 | 본 프로젝트는 인터넷상의 방대한 영상 및 댓글 데이터를 분석하여 실시간으로 밈(Meme) 트렌드를 포착하고 분석하는 시스템 구축을 목표로 한다. 단순히 유행어를 나열하는 것에 그치지 않고, 밈의 등장부터 확산 과정까지 시각화한 대시보드와 사용자 맞춤형 추천 기능을 제공한다. 특히 새롭게 등장한 밈의 유래와 구체적인 사용 맥락을 함께 제시하여 사용자가 최신 인터넷 문화의 흐름을 깊이 있게 이해하고 향유할 수 있는 통합 플랫폼을 지향한다. |
| 고객 (누구를 위해) | 빠르게 변화하는 디지털 트렌드에 민감하게 반응하며 정보 소외를 방지하고자 하는 청년층 및 일반 인터넷 사용자를 주요 대상으로 한다. 최신 유행을 콘텐츠 제작에 즉각적으로 반영해야 하는 유튜버, SNS 마케터, 커뮤니티 운영자를 핵심 고객으로 설정한다. 또한, 특정 밈의 확산 경로를 데이터를 통해 정량적으로 분석하고자 하는 전문가들에게도 실질적인 인사이트를 제공하는 전문적인 분석 도구로 활용한다. |
| Pain Point (해결할 문제) | 인터넷 밈은 생성과 소멸의 주기가 매우 짧고 여러 플랫폼에 파편화되어 존재하기 때문에, 사용자가 유행의 전체 흐름을 직접 파악하기 어렵다. 또한 특정 상황이나 영상에서 파생되는 밈의 특성상 배경 지식이 없는 사용자는 그 의미와 맥락을 이해하는 데 큰 제약을 겪는다. 이러한 정보의 휘발성과 불투명성은 사용자에게 트렌드에 대한 피로감을 유발하며, 정보 격차를 심화시키는 원인이 된다. |
| 사용 기술 | YouTube Data API를 활용하여 실시간 데이터를 확보하고, 자연어 처리(NLP) 기법과 자체 알고리즘을 통해 언급량이 급증하는 키워드를 실시간으로 감지한다. 대량의 텍스트 데이터를 안정적으로 처리하기 위해 FastAPI 기반의 백엔드 서버를 구축하며, React와 Next.js를 사용하여 직관적인 시각화 차트와 사용자 맞춤형 인터페이스를 구현한다. 이를 통해 복잡한 데이터 분석 결과를 누구나 쉽게 이해할 수 있는 시각적 정보로 변환하여 제공한다. |
| 기대 효과 | 사용자는 여러 플랫폼을 직접 확인하는 번거로움 없이 최신 인터넷 문화를 한눈에 파악할 수 있게 되어 트렌드 이해도가 비약적으로 향상된다. 콘텐츠 제작자와 마케터는 대중의 반응을 적시에 활용함으로써 콘텐츠의 경쟁력을 강화할 수 있다. 결과적으로 밈의 탄생부터 소멸까지를 데이터로 기록하고 시각화함으로써, 현대 인터넷 문화의 변천사를 객관적으로 분석할 수 있는 산업적·학술적 토대를 마련한다. |
